    ## 活动发起app
    
    1） 用户登录注册：用户进入app首页可以点击注册按钮进入注册页面，完成后使用用户信息登录
    2） 显示用户信息：左侧菜单栏显示用户名等信息
    3） 活动标题显示：用户登录后进入app首页，显示所有的活动
    4） 活动内容显示：用户点击某一个活动后会进入当前点击活动的详细页面，详细具体信息
    5） 活动发起：用户在首页可以点击右下方发起一个活动
    6） 用户可以在活动内容显示页面进行评论
    7） 用户可以在左侧菜单栏点击注销
